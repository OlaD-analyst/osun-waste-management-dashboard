# 🌍 Smart Waste & Climate Analysis in Osun State

## 📌 Overview
This project analyzes waste generation and climate trends in Osun State using data analytics and visualization.

---

## ❗ Problem
Osun State generates over 500–700 tons of waste daily.  
Improper disposal in drains and roadsides contributes to pollution and increases flood risk, especially with rising rainfall levels.

---

## 👥 Users
- Government agencies  
- Waste management authorities  
- Urban planners  
- Local communities  

---

## 💡 Solution
A data-driven dashboard built using Power BI that:

- Estimates waste generation across LGAs  
- Identifies waste hotspots  
- Integrates climate data (rainfall & temperature)  
- Highlights high-risk flood zones  

---

## 🛠 Tools Used
- Power BI  
- Excel  

---

## 📊 Data Sources
- National Bureau of Statistics (Population data)  
- World Bank (Waste generation benchmarks)  
- ResearchGate (Supporting research)  
- NASA (Climate data)  

---

## 🔍 Key Insights
- High population LGAs generate more waste  
- Waste + rainfall increases flood risk  
- Uneven waste management infrastructure  

---

## 💡 Recommendations
- Decentralized waste collection systems  
- Recycling hubs in high-waste LGAs  
- Improved drainage systems  
- Community awareness programs
